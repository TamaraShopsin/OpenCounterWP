=== Visual Form Builder Pro ===
Contributors: mmuro
Requires at least: 3.3
Tested up to: 3.3.1
Stable tag: 1.3.1

Dynamically build forms using a simple interface. Forms include jQuery validation, a basic logic-based verification system, and entry tracking.
